# SQL Business Analysis

## Project Overview

This project demonstrates how SQL can be used to answer business questions and support data-driven decision-making.

The examples focus on reporting, operational performance, customer insights, and ERP-related analysis.

---

## Business Questions

### 1. Which site generates the most ERP errors?

### 2. Which customers have missing critical information?

### 3. Which products have the highest number of data quality issues?

### 4. What are the most common error categories?

### 5. Which business units require immediate attention?

---

## SQL Skills Demonstrated

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- JOIN
- Aggregate Functions
- CASE Statements
- Data Quality Checks

---

## Business Value

SQL enables organizations to transform raw data into actionable business insights that improve reporting quality, operational efficiency, and decision-making.
